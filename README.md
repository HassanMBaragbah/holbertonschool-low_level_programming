# Holberton School - Low-Level Programming & Algorithm

This repository contains projects and exercises focused on low-level programming, structures, and algorithms as part of the curriculum at Holberton School (Tuwaiq Academy). The primary focus is to build a solid foundation in foundations of C programming, system engineering, shell scripting, and memory management.

## Project Structure

The repository is organized into distinct project directories, each targeting specific concepts:

* **`hello_world/`** - Introduction to C programming, standard I/O functions, compilation processes using `gcc`, and standard formatting/linting using Betty.

## Requirements

* **OS:** Ubuntu 20.04 LTS
* **Compiler:** `gcc` (with flags `-Wall -Werror -Wextra -pedantic -std=gnu89`)
* **Style Guide:** Betty Coding Style
