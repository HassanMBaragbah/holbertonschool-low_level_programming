# C - Variables, if, else, while

This project covers the basics of variables, conditional statements (`if`, `if...else`), and loops (`while`) in C programming. It also introduces the ASCII character set and compilation flags.

## Technologies
* **OS:** Ubuntu 20.04 LTS
* **Compiler:** gcc (using options: `-Wall -Werror -Wextra -pedantic -std=gnu89`)
* **Style Guide:** [Betty style](https://github.com/hs-hq/Betty)
