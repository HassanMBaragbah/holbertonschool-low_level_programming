# C - Functions, Nested loops

## Description
This repository contains projects and tasks designed to explore core programming concepts in the **C programming language**, specifically focusing on **functions** and **nested loops**. Functions allow us to write modular, reusable, and cleaner code by breaking down complex problems into smaller, manageable sub-tasks. Nested loops (loops inside other loops) are essential for handling multi-dimensional data structures, generating complex patterns, and working through intricate control flows.

All programs are developed and compiled on **Ubuntu 20.04 LTS** using `gcc` with strict flags (`-Wall -Werror -Wextra -pedantic -std=gnu89`) and adhere strictly to the **Betty coding style** guidelines.

## Learning Objectives
By the end of this project, I will be able to master and explain:
* What nested loops are and how to implement them.
* The difference between function declaration (prototype) and function definition.
* The importance of header files (`main.h`) and how to include them.
* Variable scope (local vs. global variables).
* Controlling execution flow without utilizing the standard C library (`printf`, `puts`, etc.), leaning heavily on a custom `_putchar` implementation.
